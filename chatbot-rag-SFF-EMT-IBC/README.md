# Mi Proyecto

Este repositorio contiene la aplicación completa, organizada en **backend** y **frontend**, junto con la configuración de Docker.

---

## Índice

### 1. Backend
Ubicación: `backend/`  
Contiene la lógica del servidor y manejo de datos.

- **scripts/** → Scripts auxiliares para tareas del servidor o datos.  
- **datos/** → Carpeta para almacenar información o bases de datos locales.  
- **server.js** → Archivo principal del servidor.  
- **package.json** → Dependencias y scripts del backend.

### 2. Frontend
Ubicación: `frontend/`  
Contiene la interfaz de usuario de la aplicación.

- **public/** → Archivos estáticos como imágenes, favicon, etc.  
- **src/** → Código fuente de la aplicación (componentes, vistas, estilos).  
- **package.json** → Dependencias y scripts del frontend.  
- **vite.config.js / webpack.config.js** → Configuración del bundler utilizado.

### 3. Docker
- **docker-compose.yml** → Configuración de contenedores para backend y frontend.

### 4. Documentación
- **README.md** → Este archivo, índice del proyecto.
